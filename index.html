🧪 What is QA?

A set of practices and policies that ensures quality in a product.

🧪 What is QA?

  • Provide Excellent customer satisfaction
  • Meet client requirements

🧐 What does it mean to have a QA Mindset?

Preventive vs Reactive

Practices of a developer with a QA mindset:

  • Thinks ahead before coding
  • Nevers leave code incomplete

  • Simple solutions == less code entropy
  • Developer is responsable fo the quality of the code

QA engineers should be ambassadors promoting the QA mindset, teaching their colleagues that tests are not only implemented later but also need to be checked through passive testing.
